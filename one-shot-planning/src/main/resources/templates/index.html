Tips

How this app behaves

  • Your anonymous user ID is stored in a durable cookie issued by the server.
  • Refreshing the browser restores your active match from persisted state.
  • Starting a new match immediately replaces any unfinished active match.
  • Every screen uses the REST API rather than embedding game rules in the page.
